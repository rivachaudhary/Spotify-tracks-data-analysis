Spotify Data Analysis & Prediction Project

Overview
This project performs end-to-end data analysis on a Spotify dataset, including data cleaning, exploratory data analysis (EDA), machine learning, and predictive modeling to understand and forecast song popularity.

What I Did
Cleaned and preprocessed raw data
Performed exploratory data analysis (EDA)
Visualized patterns in audio features
Analyzed correlations between variables
Built machine learning models
Developed a prediction system for song popularity

Models Used
Linear Regression
K-Nearest Neighbors
Random Forest
XGBoost

Key Insights
Song popularity depends on multiple audio features
Non-linear models performed better than linear models
Random Forest gave the best performance

Tools & Libraries
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
